# PR 테스트
## 🤔 요구사항
PR용 샘플로 PR을 연습합니다.
- 각자 맡은 부분의 issue를 생성합니다. (View, Domain, Model 단위로) 
- 작업용 개인 branch를 dev를 통해 생성하여 작업합니다. 
- 커밋의 단위는 작게 쪼갠 후 커밋 메시지는 상세하게 작성합니다. (말의 어미는 기/다. 로 끝내기)
- 작업이 완료되면 Dev로 PR을 생성합니다. (PR은 Git 페이지에서 작성)
- 리뷰어가 PR을 수락하면 dev에 머지됩니다.
